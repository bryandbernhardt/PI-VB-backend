## Sobre
Back-end da aplicação Charming Fox, um projeto de e-commerce desenvolvido com Node.JS e Nest para fins de estudo na faculdade de Análise e Desenvolvimento de Sistemas.

## Instalação

```bash
$ npm install
```

## Executando o app

```bash
$ npm run start:dev
```

## Executando os testes

```bash
$ npm run test:e2e
```
